**Sistema Web para la Universidad Politécnica Territorial de los Altos Mirandinos "Cecilio Acosta"**

Este proyecto consiste en el desarrollo de una plataforma web integral que automatiza y optimiza los procesos académicos y administrativos de la universidad. La plataforma incluye funcionalidades para:

- **Estudiantes**: Consulta de calificaciones, planes de evaluación, confirmación de clases y subida de certificados.
- **Profesores**: Carga de notas, gestión de horarios, subida de materiales de apoyo y clases en línea.
- **Administradores**: Automatización de procesos como concurso de ingreso, ascensos y acreditaciones.

Desarrollado con tecnologías modernas como **React.js**, **Node.js** y **MySQL**, este sistema busca mejorar la eficiencia, transparencia y experiencia de la comunidad universitaria.

---

### **Tecnologías Utilizadas**
- Frontend: React.js
- Backend: Node.js (Express)
- Base de Datos: MySQL
- Herramientas de Desarrollo: Git, GitHub, MySQL Workbench
